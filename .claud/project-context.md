# Asset Manage Agent Project

## 프로젝트 개요

AI 기반 주식 자동 투자 및 포트폴리오 관리 모바일 웹 애플리케이션

**핵심 가치:** 국내/미국 주식 통합 관리, 실시간 시세, AI 자동 매매 + 수동 거래, 투자 분석

**개발 정보:** 개발자 인범 | 시작일 2026-01-18

## 프로젝트 목적

| 항목 | 값 |
|------|-----|
| 타입 | 학습 / 포트폴리오 |
| 목표 | 기술 역량 개발 / 실험 |
| 일정 | 유연함 / 품질 중시 |

## 현재 진행 상태

| 항목 | 내용 |
|------|------|
| 상태 | 백엔드 설계 및 프론트엔드 구현 |
| 포커스 | 1단계 MVP (주식 거래 기능) |
| 현재 스프린트 | 인증 + 국내 주식 시세 조회 |

## 모듈 구성

| 모듈 | 기술 | 역할 |
|------|------|------|
| api-server | Spring Boot 4.1 + Java 21 | REST API, WebSocket, 인증 |
| web-app | Vue 3 + Vite | 모바일 웹 프론트엔드 |
| ai-agent | FastAPI + Python 3.11 | AI 분석, 자동매매 |

> 각 모듈 상세는 `{모듈}/.claud/` 참조

## 인프라

- **DB:** PostgreSQL 16 + Redis 7 + Elasticsearch 8
- **컨테이너:** Docker Compose
- **CI/CD:** GitHub Actions
- **모니터링:** Prometheus + Grafana
